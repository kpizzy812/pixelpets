# Tasks

Complete tasks to earn extra XPET rewards!

## Task Types

### Social Tasks
- Follow on Twitter/X
- Join Telegram channel
- Subscribe to YouTube
- Like and share posts

### Daily Tasks
- Log in daily
- Complete training sessions
- Invite friends

### Special Tasks
- Limited-time events
- Partnership promotions
- Community challenges

## How to Complete Tasks

1. Go to the Tasks page
2. Browse available tasks
3. Click on a task to see details
4. Complete the required action
5. Return and click "Claim"
6. XPET added to your balance!

## Task Rewards

Rewards vary by task difficulty:
- Simple tasks: 5-20 XPET
- Medium tasks: 20-100 XPET
- Special events: 100+ XPET

## Tips

- Check tasks daily for new opportunities
- Complete time-limited tasks before they expire
- Social tasks usually only need to be done once
- Some tasks may require verification
